# Five Nights at Epstein — TI-84 CE Port

A port of [FNAE](https://github.com/WitheredFOT/FNAE-Local) to the TI-84 Plus CE calculator.

## Build steps

### 1. Install prerequisites

- [CE C/C++ Toolchain](https://ce-programming.github.io/toolchain/)
- [convimg](https://github.com/mateoconlechuga/convimg)

### 2. Copy images from the original web repo

Download [FNAE-Local](https://github.com/WitheredFOT/FNAE-Local) and
copy its `assets/images/` folder into `FNAE-CE/assets/images/`.

### 3. Resize images

```bash
# Resize anything larger than 320x240 down to fit
magick mogrify -resize 320x240\> assets/images/*.png

# Office panorama should be ~480x240 for panning effect
magick assets/images/original.png -resize 480x240! assets/images/original.png
```

### 4. Convert sprites

```bash
cd assets && convimg && cd ..
```

### 5. Build

```bash
make
# Output: bin/FNAE.8xp
```

### 6. Transfer to calculator

Use **TI-Connect CE** or **TiLP** to send `bin/FNAE.8xp`.

## What changed from the web version

| Feature | Web | CE |
|---|---|---|
| Audio | `.ogg`/`.mp3` files | Removed (no speaker) |
| Video static effect | `<video>` element | `rand()` pixel scatter |
| Cutscene | HTML video | Text fade-in |
| Timers | `setTimeout`/`setInterval` | 32768 Hz hardware timer |
| Animations | CSS transitions | Tick-accumulator state machines |
| Input | Mouse + keyboard + touch | Keypad only |
| Save data | `localStorage` | CE appvar stored in archive |
| Screen | Browser window | 320×240 8bpp indexed colour |
| Images | Full-res PNGs | Resized + indexed via `convimg` |
